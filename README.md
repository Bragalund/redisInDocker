# RedisInDocker

Script for installing docker and running a redis-container.

Run on a Suse or Debian server.

## To run

```()
chmod +x runMe.sh;
./runMe.sh;
```

### To run tests

To create VM and run main.yml in root folder:
```()
vagrant up;
```

To stop VM:
```()
vagrant halt;
```

To remove VM:
```()
vagrant destroy;
```
